## Installation

1. Run `composer require bakeoff/dkfds` to install both this plugin, as well as [`dkfds/dkfds`](https://packagist.org/packages/dkfds/dkfds)
2. `cd ROOT` (where `ROOT` is path to your CakePHP application) and run `bin/cake dkfds install` to link this plugin to assets provided by dkfds/dkfds
